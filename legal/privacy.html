隐私政策

最后更新：2024年12月28日

English Version

1. 概述

Immerse（以下简称"我们"）尊重并保护您的隐私。本隐私政策说明我们如何收集、使用、存储和保护您的个人信息。使用我们的服务即表示您同意本政策。

2. 信息收集

2.1 我们收集的信息

  • 账户信息：当您登录时，我们会收集您的电子邮件地址和基本个人资料（如用户名、头像）。
  • 学习数据：您的学习统计信息（如学习词汇数量、学习时长、掌握的单词列表）。
  • 设置偏好：您的应用设置（主题、难度级别、启用的词库等）。
  • 生词本：您添加到生词本的词汇及相关上下文。

2.2 我们不收集的信息

  • 您访问的网站 URL 或浏览历史
  • 网页的具体内容
  • 您的密码（仅由第三方认证服务处理）

3. 信息使用

我们使用收集的信息用于：

  • 提供和改进我们的服务
  • 跨设备同步您的学习进度和设置
  • 生成个性化的学习建议
  • 分析服务使用情况以改进产品
  • 处理付费订阅

4. 第三方服务

我们使用以下第三方服务：

  • Supabase：用于用户认证和云数据存储
  • AI 服务提供商（OpenAI、DeepSeek、Gemini 等）：当您使用 AI 功能时，您输入的文本可能会发送至这些服务进行处理
  • 支付处理商：处理订阅付款
重要提示：当您使用自带 API Key (BYOK) 功能时，您的 API Key 仅存储在您的本地浏览器中，我们不会收集或存储这些密钥。

5. 数据存储与安全

  • 您的设置和 API Key 存储在浏览器的本地存储中
  • 云同步数据存储在安全的 Supabase 服务器上
  • 所有数据传输均使用 HTTPS 加密
  • 我们采用行业标准的安全措施保护您的数据

6. 您的权利

您有权：

  • 访问：查看我们持有的关于您的数据
  • 导出：导出您的学习数据和生词本
  • 删除：要求删除您的账户和所有相关数据
  • 撤销同意：随时停止使用我们的服务

如需行使这些权利，请通过设置页面操作或联系我们。

7. Cookie 与本地存储

我们使用浏览器的本地存储（chrome.storage）来保存：

  • 应用设置和偏好
  • 学习进度统计
  • 缓存的词典数据

这些数据存储在您的设备上，用于提供更好的用户体验。

8. 政策变更

我们可能会不时更新本隐私政策。重大变更时，我们会通过应用内通知告知您。继续使用服务即表示您接受更新后的政策。

9. 联系我们

如有任何隐私相关问题，请通过以下方式联系我们：

  • GitHub Issues: https://github.com/JHdehao/immerse-website/issues
